# Iris Flower Prediction Using Machine Learning

## Objective
To build a machine learning model that predicts the species of Iris flowers.

## Algorithm
Random Forest Classifier

## Dataset
Iris Dataset from Scikit-Learn

## Steps
1. Load dataset
2. Split data into training and testing sets
3. Train Random Forest model
4. Make predictions
5. Evaluate accuracy

## Result
The model achieved high accuracy in predicting flower species.

## Tools Used
- Python
- Scikit-Learn
- Matplotlib
- Google Colab
